README FILE
To run this code you need Python Ver 2.7.
You also need to download the PIL(Python Imaging Library) module -
the link to the site is : http://www.pythonware.com/products/pil/
To run this program you need to run the risk.py file using a python interpreter which meets the requirements specified above. 
On macs, you can run it through the terminal by typing in:  python risk.py
and pressing enter.
